# Go fmt.Printf Cheat Sheet

## Strings
- `%s` Ôćĺ string
- `%q` Ôćĺ quoted string
- `%v` Ôćĺ default

## Integers
- `%d` Ôćĺ decimal (normal int)
- `%b` Ôćĺ binary
- `%x` Ôćĺ hex (lowercase)
- `%X` Ôćĺ hex (uppercase)
- `%o` Ôćĺ octal
- `%v` Ôćĺ default

## Floats
- `%f` Ôćĺ float
- `%.2f` Ôćĺ 2 decimal places
- `%e` Ôćĺ scientific
- `%g` Ôćĺ smart shortest

## Boolean
- `%t` Ôćĺ true/false

## Structs, Slices, Maps
- `%v` Ôćĺ default
- `%+v` Ôćĺ with field names
- `%#v` Ôćĺ Go syntax

## Type & Pointer
- `%T` Ôćĺ type of variable
- `%p` Ôćĺ pointer address

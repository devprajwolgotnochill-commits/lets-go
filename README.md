# Go fmt.Printf Cheat Sheet

## Strings
- `%s` → string
- `%q` → quoted string
- `%v` → default

## Integers
- `%d` → decimal (normal int)
- `%b` → binary
- `%x` → hex (lowercase)
- `%X` → hex (uppercase)
- `%o` → octal
- `%v` → default

## Floats
- `%f` → float
- `%.2f` → 2 decimal places
- `%e` → scientific
- `%g` → smart shortest

## Boolean
- `%t` → true/false

## Structs, Slices, Maps
- `%v` → default
- `%+v` → with field names
- `%#v` → Go syntax

## Type & Pointer
- `%T` → type of variable
- `%p` → pointer address
